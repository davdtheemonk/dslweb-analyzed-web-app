About

Aim of this project is to provide a simpler UI version for people with no expertise whatseover dealing with the DAP analytics portal.Each tab on the dashboard has an info section that provides the relative info of what the tab can process or consume from the api in a layeman's language.This is to help the public get access to real-time data analysis from a simpler UI and have some capability to understand what and how MOH processes health data in KENYA.
